# Wayoo Transport Platform

Platforma łącząca pasażerów z przewoźnikami transportu grupowego.

## Hasło dostępu
`Wayoo!2025`

## Funkcjonalności

### Dla pasażerów:
- Wyszukiwanie transportu z formularzem na stronie głównej
- Tworzenie zapytań o transport
- Przeglądanie ofert od przewoźników
- Porównywanie cen i usług

### Dla przewoźników:
- Przeglądanie dostępnych zapytań
- Składanie ofert
- Zarządzanie flotą pojazdów
- Zarządzanie swoimi ofertami

## Technologie
- Next.js 15
- TypeScript
- Tailwind CSS
- Pages Router
- localStorage dla demo danych

## Development
```bash
npm install
npm run dev
```

## Deployment
Automatyczny deployment na Vercel z każdym pushem do main.
